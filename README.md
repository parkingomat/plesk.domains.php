# First Steps

start by composer

    php composer.phar start

## on linux

### install
    sh .apicra/install

### start
    sh .apicra/start

### open in browser
    sh .apicra/browser

## on windows

### install
    .apicra\install.bat

### start
    .apicra\start.bat


### open in browser
    .apicra\browser.bat
